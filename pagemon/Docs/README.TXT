pagemon
Matthew Linton
matthewl@belkin.com

Pagemon allows you to monitor the status of a single webpage using wget.

   pagemon.bat - Dowloads a single page from the specified URL and saves the page
        to the defined directory (Default is "savedpages").  Each page is numbered.

   pagemon-grind.bat - This is the same as pagemon.bat, but it does not wait any
        amount of time before fetching pages.
